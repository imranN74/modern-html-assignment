HTML V/S HTML5

HTML HTML5
Limited semantic elements, leading to the use of generic elements like "div" tag and "span" tag for layout and styling. Introduces new semantic elements such as "header" tag, nav" tag, "article" tag, "section' tag, "aside" tag, and "footer" tag to provide clearer structure to the document.
Limited support for real-time communication. Introduces the Web Sockets API, enabling full-duplex communication channels between a client and a server.
Basic form controls with limited input types. Introduces new input types like email, url, number, tel, and attributes like placeholder and required for enhanced form validation.
Limited support for multimedia; required third-party plugins (like Flash) for audio and video. Native support for "audio" and "video" elements, allowing easy embedding of multimedia content without the need for plugins.
